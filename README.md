# Trabalho PLA

Projeto de cálculo com matrizes e determinantes usando eliminação de Gauss.

## Descrição

Este trabalho implementa operações de matrizes e cálculo de determinantes em JavaScript.

## Estrutura do projeto

- `index.html` - página principal
- `styles/styles.css` - estilos do projeto
- `js/script.js` - lógica principal
- `js/service/matriz.js` - funções de matriz e determinante
- `js/ui/ui.js` - manipuladores de interface

## Como usar

Abra o `index.html` em um navegador moderno e utilize a interface para inserir dados de matrizes e calcular resultados.
